html код, основи написання

div- використовується для створення блоків на веб-сторінці.
Він може містити будь-який контент, такий як текст, зображення, таблиці, форми.
p - використовується для створення абзаців на веб-сторінці.
h1 - використовується для створення заголовків на веб-сторінці
table - використовується для створення таблиц
tr - Створює рядок в тадлиці
td- створює комірку в рядку, який був задао tr

Style
Стилі туксту і не тільки.Які бувають?

Border:ширина колір тип обведення - встворює
рамку по контурк за заданими значеннями
Border-radius: сила заукруглення в [px] –
заукруглює заукруглює грані
Colorb>: колір – >задає колір
Margin: [px] – зовнішні відступи від елемента в [px], Детальніше
Padding:[px] – внутрішній відступ елемента, задає відступ навколо вмісту елемента
